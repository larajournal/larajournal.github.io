Must-have features

Choosing the right blogging platform is crucial for the success and functionality of your blog. LaraJournal has everything you need in a blogging app.

Easy to set up

Manage users, posts, categories and tags.

Blog posts

Create and manage blog posts effortlessly with rich formatting options.

Categories

Organize content into categories for easy navigation.

Popular posts

Highlight trending articles to engage readers.

Admin panel

Control all aspects of your blog from a simple dashboard.

Roles

Define user roles for readers and admins. Coming soon

Comments and likes

Boost engagement with interactive comments and likes.

Post`s visual editor

Easily design posts with a visual content editor. Coming soon

How to set up?

Just clone the project via git-command:

                git clone git@github.com:larajournal/larajournal.git

And follow the instructions from https://github.com/larajournal/larajournal
